# at-site-core

Plugin for Gatsby.

It generates:

- from Markdown - pages
- from YAML- main menu, social links, footer nav, address, contacts

## Plugin options

| Name           | Default value     | Note
|---             |---                | ---
| pageDirs       | { page: 'pages' } |
| i18n           | REQUIRED: object  |
| noIndex        | false             | if true force `noindex` field to true
